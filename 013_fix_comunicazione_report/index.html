TASK 013

Fix comunicazione report

P0 · Size S · Fix

Il report usa linguaggio troppo tecnico. Va reso comprensibile per un direttore sanitario che non ha mai visto ClinicalScenario. Niente sigle, niente valori statistici grezzi, solo etichette chiare in italiano.

Rendere il report leggibile da chiunque senza bisogno di spiegazioni.

Cosa fare

  1. Sostituire "item" con "domande" ovunque
  2. Sostituire valori SD con etichette qualitative (alta variabilita, gruppo diviso, ecc.)
  3. Pill comprensibili: "Gap formativo", "Domanda debole", "Allineato" - non codici
  4. Colonna "Qualita domanda" al posto di "Discriminazione" con pill buona/debole/inadeguata
  5. Aggiungere un box introduttivo prima dei risultati che spiega cosa si sta guardando
  6. Titoli metodo in italiano descrittivo

Criteri di accettazione

  • ✓ Nessun termine tecnico visibile nel report
  • ✓ Pill tutte in italiano comprensibile
  • ✓ Box introduttivo presente prima dei risultati
  • ✓ Report leggibile senza spiegazioni aggiuntive
