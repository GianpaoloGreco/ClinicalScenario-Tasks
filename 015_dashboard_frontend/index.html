TASK 015

Report visuale

P1 · Size L · Feature

I risultati dell'analisi vanno presentati come dashboard interattiva. Due modalita: Sintesi (2 pagine per chi ha fretta) e Approfondita (7 sezioni navigabili). Export PDF con un click.

Costruire la dashboard con doppia modalita (Sintesi/Approfondita), profili individuali cliccabili e export PDF.

Cosa fare

  1. Dashboard con sidebar, due modalita (Sintesi/Approfondita), navigazione tra sezioni
  2. Sintesi: risultati chiave + panoramica preferenze in 2 schermate
  3. Approfondita: 7 sezioni (guida lettura, intestazione, preferenze, competenza, crosscut, profili, appendice)
  4. Click su partecipante apre profilo individuale con posizione su tutti gli assi
  5. Export PDF via stampa browser con CSS A4
  6. Tutti i testi seguono le regole del Task 013 (niente tecnicismi)

Criteri di accettazione

  • ✓ Dashboard funzionante con dati reali
  • ✓ Sintesi in 2 pagine, Approfondita in 7 sezioni
  • ✓ Profilo individuale cliccabile per ogni partecipante
  • ✓ PDF esportabile con layout A4
  • ✓ Responsive sopra 680px
